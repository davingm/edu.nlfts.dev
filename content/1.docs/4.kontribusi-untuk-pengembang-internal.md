Repositori ini menggunakan **monorepo**. Seluruh aplikasi edu berada di root project, sedangkan seluruh konten dokumentasi (Markdown, konfigurasi konten, dan data dokumentasi lainnya) berada di dalam folder `www/`.

## Prasyarat

::steps
  ### Pastikan Anda Memiliki Akses Repo
  Anda harus memiliki akses pada repository [developer](https://github.com/nairha/developer) dan [edu.nlfts.dev](https://github.com/nlfts/edu.nlfts.dev). Jika belum memiliki akses,
  silakan hubungi maintainer lewat from issue [di sini](https://github.com/nlfts/edu.nlfts.dev/issues). 

  ::alert{type="success" icon="lucide:lightbulb"}
  pengajuan akan di nilai dari keaktifan dan kontribusi Anda pada proyek organisasi
  :: 
  ### Pastikan sudah menginstal

  * Node.js versi terbaru saat ini ([v24.18.0](https://nodejs.org/id/download))
  * PNPM versi terbaru ([v11.x](https://pnpm.io/installation))
  * Git
  * Powershell / Bash (*Kami Lebih Menyarankan anda menggunakan bash*)
::

---

## 1. Langkah Step Awal

::steps
  ### 1. Fork Repository Utama

Buka repository berikut:

[https://github.com/nairha/developer](https://github.com/nairha/developer)

Kemudian lakukan **Fork** ke akun GitHub Anda.

  ### 2. Clone Repository

Clone repository hasil fork Anda ke local komputer:

```bash [terminal]
git clone https://github.com/<username>/developer.git
cd developer
```

Struktur project akan terlihat seperti berikut:

```text [txt]
.
├── app/
├── assets/
├── components/
├── composables/
├── Dev/
├── i18n/
├── lib/
├── pages/
├── plugins/
├── public/
├── scripts/
├── server/
├── types/
├── www/
├── www-backups/
├── app.vue
├── build.log
├── components.json
├── eslint.config.js
├── LICENSE
├── nuxt.config.ts
├── package.json
├── pnpm-lock.yaml
├── pnpm-workspace.yaml
├── README.md
├── sample.career.md
├── taze.config.ts
├── tsconfig.json
└── vercel.json
```

 ### 3. Fork Repository Konten Dokumentasi

Folder `www/` digunakan untuk menyimpan seluruh konten dokumentasi, seperti:

* Markdown
* Konfigurasi dokumentasi
* Konten halaman
* Data dokumentasi lainnya

Masuk ke folder `www`:

```bash [terminal]
cd www
```

Kemudian buka repository berikut:

https://github.com/nlfts/edu.nlfts.dev

Lakukan **Fork** ke akun GitHub Anda.

Setelah itu clone repository hasil fork ke dalam folder `www`:

```bash [terminale]
git clone https://github.com/<username>/edu.nlfts.dev.git .
```
::alert{type="note" icon="lucide:pencil"}
  Gunakan tanda titik (`.`) di akhir perintah agar isi repository langsung ditempatkan ke dalam folder `www`.
::

::

## 2. Installasi Proyek

  Setelah anda mengikuti langkah langkah di atas silakan Kembali ke root project:

  ```bash [Terminal]
    cd ..
  ```

  Kemudina lakukan Install seluruh dependency:

 ::code-group
    ```bash [npm]
    npm install
    ```
    ```bash [pnpm]
    pnpm install
    ```
    ```bash [bun]
    bun install
    ```
  ::
